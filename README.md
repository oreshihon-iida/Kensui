# Kensui - シンプルな懸垂記録アプリケーション

## アプリケーション概要
Kensuiは、日々の懸垂トレーニングを簡単に記録・管理できるクロスプラットフォームアプリケーションです。

### 主な機能
- 懸垂回数の手動入力と記録
- 日別合計回数のグラフ表示
- トレーニング履歴の管理

### 技術スタック
- Flutter/Dart（クロスプラットフォーム開発）
- Material Design（UIコンポーネント）
- SharedPreferences（ローカルストレージ）
- fl_chart（グラフ表示）
